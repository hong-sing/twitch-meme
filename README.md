# 🎮 트위치 밈 저장소

* 트위치의 밈(meme)에 대한 게시글을 작성할 수 있는 사이트
* https://twitch-meme.shop

## 사용기술
* Java 11
* SpringBoot 2.7.11
* Gradle
* MariaDB
* Spring Security
* Spring Data JPA
* Thymeleaf
* JQuery
* AWS
  * EC2
  * S3
  * RDS
  * CodeDeploy
* Github Actions
* Nginx

## 주요 기능
1. 게시글 기능
2. 댓글 기능
3. 로그인 및 회원탈퇴
4. 좋아요 기능
5. 신고 기능
6. 스트림 조회 기능
7. 자신이 팔로우 한 스트리머 확인 기능
8. 특정 스트리머가 팔로우 한 채널 확인 기능
